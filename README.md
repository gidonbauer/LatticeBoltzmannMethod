# Lattice Boltzmann Method

An implementation of the [Lattice Boltzmann Method](https://en.wikipedia.org/wiki/Lattice_Boltzmann_methods).
Work in progress.

## Quickstart

```console
$ cmake -Bbuild
$ cd build
$ make
$ ./main
```

## Dependencies

The dependencies are managed via git-submodules.
HDF5 is optional and must be installed separately.

- [OpenMP](https://www.openmp.org/)
- [Igor](https://github.com/gidonbauer/Igor)
- [HDF5](https://github.com/HDFGroup/hdf5)
